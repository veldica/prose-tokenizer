# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-23

### Added
- Initial release of `@veldica/prose-tokenizer`.
- Deterministic rule-based tokenization for paragraphs, sentences, and words.
- Markdown-aware structure detection (headings, list items, blockquotes).
- Character metrics (character count, character count no spaces, letter count).
- Zero-dependency ESM package with TypeScript types.
- Alias `tokenizeProse` for the primary `tokenize` function.
